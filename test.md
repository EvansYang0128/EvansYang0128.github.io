# 教學指南

## 介紹
這個指南將帶您一步步完成專案的安裝和運行。

## 先決條件
- 確保您已安裝 Node.js。您可以從 [nodejs.org](https://nodejs.org/) 下載。

## 步驟

1. **克隆儲存庫：**
    ```sh
    git clone <repository-url>
    ```

2. **進入專案目錄：**
    ```sh
    cd <project-directory>
    ```

3. **安裝依賴項：**
    ```sh
    npm install
    ```

4. **構建專案：**
    ```sh
    npm run build
    ```

5. **運行專案：**
    ```sh
    npm start
    ```
